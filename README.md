# 🚀 Levitar

Site institucional desenvolvido com HTML e CSS puro, focado em uma identidade visual moderna e navegação fluida entre múltiplas páginas.

## 📋 Sobre o Projeto

O **Levitar** é uma aplicação web estática que apresenta um serviço ou marca com temática de levitação/flutuação. O projeto conta com diversas páginas completas — desde cadastro e feedback até apresentação da equipe e um guia do produto — demonstrando uma estrutura de site completa feita do zero.

## 🖥️ Tecnologias Utilizadas

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)

- **HTML5** — estrutura e semântica das páginas
- **CSS3** — estilização e layout responsivo

## 📁 Estrutura do Projeto

```
Levitar/
├── index.html              # Página principal
├── equipe.html             # Apresentação da equipe
├── guia.html               # Guia do produto/serviço
├── cadastre.html           # Página de cadastro
├── obrigado-cadastre.html  # Confirmação de cadastro
├── feedback.html           # Formulário de feedback
├── obrigado-feedback.html  # Confirmação de feedback
├── estilo.css              # Estilos principais
├── style.css               # Estilos complementares
└── assets/                 # Imagens e recursos visuais
```

## ✨ Funcionalidades

- Página inicial com apresentação do produto
- Formulário de cadastro de usuários
- Sistema de feedback com confirmação
- Página de apresentação da equipe
- Guia completo do produto
- Design visual coeso com identidade própria

## 🚀 Como Executar

Por ser um projeto estático em HTML/CSS, basta abrir o arquivo `index.html` em qualquer navegador moderno:

```bash
# Clone o repositório
git clone https://github.com/bryanwinchezz/Levitar.git

# Acesse a pasta do projeto
cd Levitar

# Abra o arquivo no navegador
open index.html
```

Ou arraste o arquivo `index.html` diretamente para o seu navegador.
